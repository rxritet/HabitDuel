Place your permanent app icon here:

- File name: `app_icon.png`
- Recommended size: `1024x1024`
- Format: PNG with transparent background if needed

After adding the file, run:

```powershell
C:\flutter\bin\flutter.bat pub get
C:\flutter\bin\dart.bat run flutter_launcher_icons
```

This will regenerate the Android and iOS launcher icons from one source image.
